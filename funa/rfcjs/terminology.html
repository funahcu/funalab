association
アソシエーション。
chunk type
チャンク型。チャンクの区別をつけるための数字。
SCTP endpoint
SCTP端末。端点でもいいのかな。
data sender
データ送信者。あるデータについてデータを送信する側のSCTP端末。
data receiver
データ受信者。あるデータについてデータを受信する側のSCTP端末。
ULP (Upper Layer Protocol)
上位層プロトコル
primitive
基本要素。プリミティブ。RFC2960の10章では抽象関数の形で扱われている。
stream sequence number
ストリームシーケンス番号。各ストリームでながれるデータチャンクのそれぞれにつけられる通し番号。
operational error
操作エラー。処理中に解釈できないチャンクに出合った場合など。RFC2960の3.3.10節参照。
fast retransmit
高速再送(RFC2960J、RFC3758J)、早期再送(RFC4460J) ... あとで統一
Gap Ack Block
ギャップAckブロック(RFC2960JではGap Ack Blockのまま)。シーケンス番号であるTSNが歯ぬけの状態でところどころ未到達の場合、部分的に順序通り届いている番号の先頭と最後を組にして、確認応答するためのもの。
Cumulative TSN Ack
累積TSN確認応答。TSNは順序通り届けるべきでありそれを確認するため、最初からどこまでなら順序通りに届いているかを示す。この番号より大きなTSNをもったブロックは歯ぬけの状態で届いているか全く届いていないかである。
bundle
バンドル。ひとつのIPパケットの中に複数のチャンクを入れること。バンドリングとも。
verification tag
認証タグ。
initiate tag
初期化タグ。
silently discard
静かに破棄する。特にメッセージなどは送出しない。
